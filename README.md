### 💱 Spotlight: ZAR/USD Exchange Rate Prediction — Regression Exam Project

> **Dataset:** `rand-dollar.csv` | **Model:** Simple & Multiple Linear Regression | **Split:** 80/20 (no shuffle)

This exam project involved building a linear regression model to predict the South African Rand to US Dollar exchange rate using macroeconomic export data, then evaluating and interpreting the model with statistical rigour.

**Setup & Pipeline**
```python
import pandas as pd
from sklearn.model_selection import train_test_split
from sklearn.linear_model import LinearRegression

df = pd.read_csv('rand-dollar.csv', index_col=0)

# Features and target
X = df.drop(columns='ZAR/USD')
y = df['ZAR/USD']

# 80/20 split — chronological order preserved (shuffle=False)
X_train, X_test, y_train, y_test = train_test_split(X, y, test_size=0.2, shuffle=False)

# Train on a single predictor: Value of Exports (ZAR)
lm = LinearRegression()
lm.fit(X_train[['Value of Exports (ZAR)']], y_train)
```

**Key Findings & Interpretations**

| # | Question | Answer |
|---|----------|--------|
| 1 | **Intercept interpretation** | ZAR/USD equals the intercept value when exports are zero — the baseline exchange rate with no export activity |
| 2 | **Slope value** | `8.68e⁻⁵` — a very small but positive coefficient |
| 3 | **Slope interpretation** | A 1-unit increase in exports results in an increase of `8.68e⁻⁵` units in the ZAR/USD exchange rate |
| 4 | **Predicted rate at R100,000 exports** | **R11.97 / $1.00** |
| 5 | **MSE on test set** | `8.22` |
| 6 | **R² on test set** | `-8.45` |
| 7 | **Negative R² implication** | The model performs *worse* than a simple horizontal line at `y = mean(y)` — exports alone are a poor predictor of the exchange rate on unseen data |
| 8 | **Predicted rate for August 2017** | **R12.25 / $1.00** |
| 9 | **Absolute error for August 2017** | **R0.98** |

**Correlation Analysis (Full Dataset)**

Using `df.corr()` to identify the strongest and weakest predictors of ZAR/USD:

| Relationship | Variable |
|---|---|
| 📉 **Weakest linear relationship** with ZAR/USD | **Savings Rate** |
| 📈 **Strongest linear relationship** with ZAR/USD | **Consumer Price Index (CPI)** |

**Takeaways**
- A single-predictor model using export values yielded a **negative R²** on the test set, confirming it generalises poorly — a key lesson in why feature selection and multivariate modelling matter.
- CPI emerged as the strongest macroeconomic predictor of the exchange rate, suggesting inflation dynamics drive currency valuation more than trade volumes in this dataset.
- This project reinforced the importance of **model evaluation beyond training accuracy** — a model can fit training data reasonably but fail entirely on held-out data.
